生活如诗，厚积薄发

Posted on 2017-03-05

最近，过得有点辛苦。连续一个月，我全身心在现在做的这个方法上面，但是最后发现效果并没有想象中的好。我不禁开始思考，问题出在哪里。

最近组里面中了4篇CVPR，一个计算机视觉领域的顶级会议，如果我想要毕业，中一篇可能就差不多了。所以，我们组的目标一直都是以发paper为首要。我不知道别的组是什么个情况，但是通常学校和机构，大都是以发paper为目的。可以看出来，发paper是对自己科研成果的展示，是体现自己的水平以及组里面的水平的重要方式。

在知道现在自己做的方法有机会发paper时，我当然是高兴的。到了方法失败，我当然是沮丧的。沮丧之余，我便在思考，我的失败是偶然，还是必然？

能在那些顶会上面发paper的都是些牛人，大多是某研究机构的scientists，这些scientists都是某校已经毕业的phd，或者在读的phd，像我这样刚开始读master的在极少数。如果仅仅把这些当作失败的理由，你会觉得我妄自菲薄。我对比这个是为了说明一个事情，就是那些phd或者已经毕业的phd，比我更多的东西是，他们的积累。这样一想来，就能够感受到问题所在了。以我现在的能力，即使发了paper，毕了业，我到底掌握的东西会有多少？少！少得可怜！学校以至少4年的时间培养出一个phd，为的是让他们在这段时间用心积累，成为领域上的专家。所以，对领域的了解与认识是基础，有所创新是更进一步。如果你不是领域上的专家，即使发了paper，意义也不太大。反观我自己，一个还不到一年的master，就想一步登天。处在一个本末倒置的情况，盲目追求创新而忽略积累，所以有这样的失败，是必然的。

想通了这些，就好受多了，自然接下来的事情就是积累，然后厚积薄发。

厚积薄发肯定跟耐心有关。而将自己得生活活得耐心，便是要享受自己的生活，享受当下的一切。这便是，生活如诗。生活如一首诗，句句用心，字字可品可斟酌。生活也如写诗，读书破万卷，下笔才如有神。去其糟粕，取其精华，也是诗的精妙之处。将生活中有意义的事情拿起来，细细品味，感受那些小确幸，感受生活的充实与幸福，也要把那些没意义的事情抛弃掉，做到断舍离。

我先谈了厚积薄发，再谈生活如诗，是从方法到本质。事实上，从本质到方法便是：生活如诗，厚积薄发。
